1) Palindrome check
- Assignment name: verify_mirror
- Expected files: verify_mirror.py
- Allowed functions: len, .lower, .upper
- Write a function:
  def verify_mirror(text: str) -> bool
- Input: A string text.
- Output: Boolean indicating if text is a palindrome when case-insensitive, without removing spaces or special characters.
- Examples:
  verify_mirror("level") -> True
  verify_mirror("Level") -> True
  verify_mirror("hello") -> False

- Assignment name: reverse_text
- Expected files: reverse_text.py
- Allowed functions: len
- Write a function:
  def reverse_text(text: str) -> str
- Input: A string text.
- Output: The reversed string.
- Examples:
  reverse_text("abc") -> "cba"
  reverse_text("rats") -> "star"

- Assignment name: merge_and_sort
- Expected files: merge_and_sort.py
- Allowed functions: none restricted
- Write a function:
  def merge_and_sort(list_a: list, list_b: list) -> list
- Input: Two lists of numbers (or comparable items).
- Output: A new list containing all elements from both lists, sorted ascending.
- Examples:
  merge_and_sort([3, 1], [2, 4]) -> [1, 2, 3, 4]
  merge_and_sort([5], [3, 3, 1]) -> [1, 3, 3, 5]

- Assignment name: shift_letters
- Expected files: shift_letters.py
- Allowed functions: none restricted
- Write a function:
  def shift_letters(text: str, shift: int) -> str
- Input: text (string) and shift (int, positive or negative).
- Output: Text with letters shifted by the given amount; non-letters unchanged; wrap within A-Z and a-z separately.
- Examples:
  shift_letters("abc", 2) -> "cde"
  shift_letters("XYZ", 2) -> "ZAB"
  shift_letters("ab-cd", -1) -> "za-bc"

- Input: decimal (non-negative int), base (2..16).
- Output: Representation of decimal in the given base as a string (use digits 0-9 and A-F). decimal = 0 yields "0".
- Examples:

8) Right rotation (train cars)
- Assignment name: rotate_right
- Expected files: rotate_right.py
- Allowed functions: none restricted
- Write a function:
  def rotate_right(lst: list, n: int) -> list
- Input: lst (list of elements), n (non-negative).
- Output: New list after a right rotation by n positions (last n elements move to the front). Use the logic: cut = len(lst) - n; tail = lst[cut:]; head = lst[:cut]; return tail + head
- Examples:
  rotate_right([1,2,3,4,5], 2) -> [4,5,1,2,3]
  rotate_right(["a","b","c"], 1) -> ["c","a","b"]

- Examples:
  multiset_intersection("aabbcc", "abbbc") -> "aabbc"
  multiset_intersection("cat", "cta") -> "cat"
